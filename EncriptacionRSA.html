Encriptación en RSA

Descubre el poder del inverso modular

En Aritmética Modular, dos números enteros a y b son considerados congruentes con un modulo c si, al dividir ambos por n, dejan el mismo residuo.

Esto significa que a diferencia entre a y b es un multiplo de c, es decir, a - b = Kn para algún número entero k. La congruencia modular es clave en muchas ramas de las matemáticas y tiene aplicaciones importantes encriptografía, como el algoritmo RSA.

¿Como se calcula?

El proceso de encriptación RSA comienza generando dos números primos p y q, a partir de los cuales se calcula n = p ⋅ q y ϕ(n), que es el producto de (p − 1) y (q − 1). Luego, se elige un número e que sea coprimo con ϕ(n), y se obtiene su inverso modular d mediante el algoritmo extendido de Euclides.

La clave pública está compuesta por e y n, mientras que la privada por d y n. Para encriptar un mensaje m, se usa la fórmula c = m^e mod n. El destinatario descifra el mensaje cifrado c con su clave privada mediante m = c^d mod n, haciendo que sin la clave privada, descifrar el mensaje sea muy difícil.
